This release contains internal improvements.

Versions 0.6.3 thru 0.6.7 are unstable intermediate releases and should not be used.

{/* truncate */}

## Alpine Base Image

The backend image is now based on Alpine to reduce the number of unnecessary dependencies and image size.

## Removed Postgres Chart Dependency

Following [Bitnami's deprecation notice](https://github.com/bitnami/charts/issues/35164) the Postgres dependency in old charts can't be used anymore.

Please install Postgres separately from Platz.
