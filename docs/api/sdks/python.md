# Python SDK

The Python SDK is published to PyPI as
[`platz`](https://pypi.org/project/platz/) and is generated from the
backend's OpenAPI schema, versioned to match the backend release.

```bash
pip install platz
```

Pre-release (beta) versions install with the `--pre` flag:

```bash
pip install --pre platz
```
